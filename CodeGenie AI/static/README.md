# 🚀 CodeGenAI

An AI-powered web application that generates code in multiple programming languages with explanations and user tracking.

---

## 📌 Project Overview

CodeGenAI helps users generate clean code from simple prompts. It supports multiple languages and provides explanations along with user tracking.

---

## ✨ Features

- User Login System  
- AI Code Generation  
- Multi-language Support  
- Syntax Highlighting  
- Code Explanation  
- User History Tracking  
- Admin Dashboard  
- Feedback System  

---

## 🧠 Technologies Used

- Frontend: HTML, CSS  
- Backend: Python (Flask)  
- Database: SQLite  
- AI: Groq API  

---

## ⚙️ How It Works

1. User logs in  
2. Selects language  
3. Enters prompt  
4. Code is generated  
5. Explanation is shown  
6. Data stored in database  

---

## 🚀 Run Project

pip install -r requirements.txt  
python app.py  

Open: http://127.0.0.1:5000
